# ZinZin Furry Wiki

Static lore/wiki website for `Furry: Paws in Void's Tail`.

## Files

- `index.html` - main page
- `styles.css` - full responsive styling
- `app.js` - mobile nav and wiki search
- `assets/images/` - copied game images used by the page
- `locales/languages.json` - language manifest used by the dropdown
- `locales/vi.json` - Vietnamese text keys

## Deploy

Upload the whole `zinzin_furry` folder to any static host.

To add a new language, create another JSON file in `locales/`, then add it to
`locales/languages.json`.
